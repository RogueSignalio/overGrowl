OverGrowl by RogueSignal.IO
Simple, clean and flexible "growl" style notification component.
  • Default images and style all embedded in one JS file for instant use via one file.
  • Timing options for fade and duration & programmatic style overrides.
  • Expand with new types beyond alert,error,info,success,system and basic.
  • Prevents (by default) duplication of messages in view.
  • HTML content enabled. Glint visual on icon or body to draw attention to growl.
  • Option for text auto select and auto copy to clipboard with failover.
  • Close on panel click, close X button, and/or timed close.
  • Floats left, right, or embed in element and sticks to viewport top.
  • Banner modes, embed in elements, float left or right over UI, and share multiple instances in one element, but with different notice tyles.
  • And much more - View this source for these examples.
